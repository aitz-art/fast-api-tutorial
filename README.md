# fast-api-tutorial

## setup

```bash
pip install pandas
pip install numpy
pip install sklearn
# pip install pickle # standard lib python
pip install FastAPI
pip install "uvicorn[standard]"
pip install requests
pip install pydantic
```

## Create VENV

```bash
python3 -m venv fast-api-venv
source fast-api-venv/bin/activate
```

## Test

```bash
# TEST SCRIPTS
uvicorn app.main:app 
```
